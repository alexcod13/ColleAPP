Génère une colle sur mesure avec une IA à partir de ton cours, importe-la ici et entraîne-toi en conditions réelles avec correction immédiate.

Étape 01 Copier le prompt IA

Clique ci-dessous pour copier le prompt dans ton presse-papier. Colle-le dans une IA générative (Claude, ChatGPT, Gemini…) avec ton cours en pièce jointe. L'IA te génèrera automatiquement un fichier JSON prêt à l'emploi.

💡 Astuce : sur Claude, envoie le prompt + glisse-dépose ton PDF de cours. Demande-lui de générer 20 à 30 questions difficiles et transversales.
Étape 02 Coller le JSON généré

Copie le JSON renvoyé par l'IA et colle-le ci-dessous. Assure-toi que le texte commence par { et se termine par }.
